About Image Upscaling

This tool increases the resolution of your images using basic canvas resizing with high-quality image smoothing enabled. While it makes images larger, it's not a true AI super-resolution upscaler.

For best results, start with a good quality original image. The output format will be PNG for most inputs, or JPEG if the original was a JPEG, to preserve quality where possible.

Maximum upscaled dimension is capped to prevent browser performance issues.
